# Data Scientist Portfolio - Visual Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep navy blue (#1a365d) - Professional, trustworthy, tech-focused
- **Secondary**: Warm amber (#f6ad55) - Energy, innovation, data insights
- **Accent**: Soft teal (#4fd1c7) - Modern, analytical, complementary
- **Neutral**: Light gray (#f7fafc) - Clean background, readability
- **Text**: Charcoal (#2d3748) - High contrast, professional readability

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for content and UI elements
- **Code Font**: "JetBrains Mono" - Monospace for technical content and code snippets

### Visual Language
- **Minimalist Professional**: Clean layouts with purposeful whitespace
- **Data-Driven Aesthetic**: Subtle geometric patterns and data visualization elements
- **Modern Tech**: Contemporary design patterns that reflect technical expertise
- **Accessible**: High contrast ratios and clear visual hierarchy

## Visual Effects & Styling

### Used Libraries & Effects
1. **Anime.js**: Smooth scroll animations, card hover effects, text reveals
2. **Typed.js**: Typewriter effect for hero section tagline
3. **Splitting.js**: Character-by-character text animations for headings
4. **ECharts.js**: Interactive data visualization for skills and project metrics
5. **Splide.js**: Smooth project showcase carousel
6. **p5.js**: Animated background particles representing data points
7. **Pixi.js**: Subtle shader effects for banner background

### Animation Strategy
- **Hero Section**: Typewriter animation for main tagline with gradient color cycling
- **Project Cards**: 3D tilt effect on hover with shadow expansion
- **Skills Section**: Animated progress bars with staggered reveals
- **Background**: Subtle floating particles representing data visualization
- **Navigation**: Smooth color transitions and underline animations

### Header & Hero Effects
- **Banner Background**: Animated gradient mesh using Pixi.js shaders
- **Profile Image**: Circular mask with subtle glow effect on hover
- **Typography**: Large serif headings with character stagger animations
- **Call-to-Action**: Buttons with morphing backgrounds and icon animations

### Interactive Elements
- **Project Cards**: Dual-action buttons (GitHub + Demo) with icon transitions
- **Skills Filter**: Animated category buttons with active state indicators
- **Contact Form**: Floating labels with validation state animations
- **Navigation**: Smooth scroll-to-section with progress indicators

### Responsive Design
- **Mobile-First**: Optimized for high-resolution displays
- **Flexible Grid**: CSS Grid and Flexbox for adaptive layouts
- **Touch-Friendly**: Appropriate button sizes and spacing for mobile
- **Performance**: Optimized animations that respect reduced motion preferences

### Data Visualization Theme
- **Consistent Color Mapping**: All charts use the defined color palette
- **Interactive Elements**: Hover states and tooltips for data exploration
- **Clean Aesthetics**: Minimal chart junk, focus on data clarity
- **Responsive Charts**: Adaptive sizing for different screen sizes

### Professional Polish
- **Micro-Interactions**: Subtle feedback for all user actions
- **Loading States**: Skeleton screens and progress indicators
- **Error Handling**: Graceful degradation and user-friendly messages
- **Accessibility**: ARIA labels, keyboard navigation, screen reader support
